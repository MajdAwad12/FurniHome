# FurniHome - Furniture Store Prototype

FurniHome is a modern responsive furniture catalog website built with React, Vite, and Tailwind CSS.

The website is designed as an initial prototype for a furniture store. It allows customers to browse furniture products by category, view product codes, switch between multiple languages, and contact the store through WhatsApp.

## Features

- Modern responsive homepage
- Furniture product catalog
- Product categories filtering
- Product code for each item
- Favorite heart button for products
- WhatsApp contact button for each product
- Floating WhatsApp button
- About section
- Contact section
- Multi-language support:
  - Hebrew
  - English
  - Arabic
- RTL and LTR layout support
- Environment variable support for WhatsApp phone number

## Technologies Used

- React
- Vite
- Tailwind CSS
